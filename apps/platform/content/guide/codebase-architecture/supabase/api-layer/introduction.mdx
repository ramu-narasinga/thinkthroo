In this guide, you will learn the API layer used in [Supabase](https://github.com/supabase/supabase). 

Supabase is the open source Firebase alternative. Supabase gives you a dedicated Postgres database to build your web, mobile, and AI applications.

# What's API Layer?
In the context of web applications, API layer means communicating with your server to get/mutate data.
Inspired by [Bulletproof React's API Layer](https://github.com/alan2207/bulletproof-react/blob/master/docs/api-layer.md)
We analyze the API Layer in the [Supabase](https://github.com/supabase/supabase). 

# Supabase API Layer
To study the API Layer, we pick a page on https://supabase.com and look at the source code except we are
narrowing down our expedition to focus on the API Layer.

# Concepts you will learn:
