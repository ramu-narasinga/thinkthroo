[L: Root level structure]
Let's look at an overview of the project structure of [Shadcn/ui at root level](https://github.com/shadcn-ui/ui).

```sh
apps/www               # `apps/www` folder is workspace.
|
packages               # `packages` folder contains two folders - cli, shadcn.
|
scripts                # `scripts` has only one file named sync-templates.sh.
|
templates              # `templates` folder contains monorepo-next template.
```

> Only folders are listed here, for configuration details please check out Tooling in Shadcn/ui
