# **Installation**

To get started with Nue, you will have to follow the below steps:

1. Install Bun
    

```javascript
# Install Bun
curl -fsSL https://bun.sh/install | bash
```

2\. Install Nue

```javascript
# Install Nue
bun install nuekit --global
```

3\. Install app

```javascript
nue create simple-blog # This creates a simple blog
# or
nue create simple-mpa # This creates a multi-page application
```

4\. View the application at [http://localhost:8083/](http://localhost:8083/)

Read more about Nue.js in their [documentation](https://nuejs.org/docs/).
